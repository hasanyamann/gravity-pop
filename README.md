# Gravity Pop

A puzzle game where you can't move the balls — you rotate the whole board!

**Play now:** https://hasanyamann.github.io/gravity-pop/

## How to play

- Swipe left/right (or tap the rotate buttons) to rotate the board 90°
- All balls fall with gravity
- 3 or more same-colored balls touching = pop!
- Chain reactions build combos for bonus points
- Reach the goal before you run out of moves

## Features

- Endless procedurally generated levels
- Two goal types: pop any balls, or pop a specific color
- Star ratings (1–3 stars per level)
- Combo system with chain reactions
- Sound effects and haptic feedback
- Works offline, installable as an app (PWA)

## Tech

Single-file HTML5 canvas game. No frameworks, no build step.
